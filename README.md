### E-Shop API
#### Features:
    * User Resister/Login
    * Getting Category tree
    * Getting Product list, also with filtering by category, price, attributes
    * Getting Product by id or slug
    * Cart CRUD
    * Creating Order
    * Getting Order list for authorized User
